## Skills

- **Frontend Development**: React, TypeScript, JavaScript, Vite, Responsive Design
- **Backend Development**: Node.js, TypeScript, RESTful APIs
- **CMS & Content Management**: Strapi CMS
- **Database Management**: Database design and migrations
- **Web Standards**: HTML5, CSS3, ES6+

## Features

- **Responsive Design**: Fully responsive across all devices
- **Modern Stack**: Built with React and powered by Strapi CMS
- **Type Safety**: TypeScript for robust development
- **Fast Performance**: Vite for optimized builds
- **Content Management**: Easy content management through Strapi
- **Organized Structure**: Modular architecture for scalability

## Getting Started

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
npm run develop
```

## API Endpoints

The backend provides the following main resources:
- `/projects` - Portfolio projects
- `/skills` - Professional skills
- `/categories` - Project categories
